# House Price Prediction

## Overview
This project builds a simple machine learning model to predict housing prices using the California Housing dataset from scikit-learn.

It demonstrates an end-to-end workflow including data preprocessing, model training, and evaluation.

---

## Objectives
- Practice a complete machine learning pipeline
- Apply regression techniques to real-world data
- Evaluate model performance using standard metrics

---

## Tech Stack
- Python
- Pandas
- scikit-learn

---

## Workflow
1. Load dataset from scikit-learn  
2. Perform data preprocessing (handling missing values, scaling)  
3. Split data into training and testing sets  
4. Train a Linear Regression model  
5. Evaluate performance using RMSE and cross-validation  

---

## Results
- Model evaluated using Root Mean Squared Error (RMSE)  
- Cross-validation used to ensure model stability
